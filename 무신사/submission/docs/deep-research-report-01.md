# 무신사(MUSINSA) — 채택 후보 근거 요약

> **정보 시점**: 2026년 7월 8일 기준 공개 자료(무신사 공식 IR·뉴스룸, 공정거래위원회 조치 보도 등). 규제 사안은 2026년 4월 보도까지 반영.

## 채택 후보: 입점 계약서 리스크 조항 스캐너 `musinsa-contract-risk-scanner`

> ✅ **채택 (제출용 최종 선정, 2026-07-08)** — 구현 스켈레톤 위치: 워크스페이스 루트 `submission/`. 선정 근거: 공정위 조사(2024.8 멀티호밍·2026.4 직매입)라는 확신도 '상' 공개 근거 + 계약서 텍스트라는 명확한 입력 + 샘플 fixture만으로 재현 가능한 검증.

### 해결 대상 문제 (근거)

- **문제 3-2 — 멀티호밍 제한·최혜대우(MFN) 요구 의혹**: 공정위가 2024년 8월 26일 무신사 본사에 조사관을 보내 입점 브랜드 계약서를 확보하는 현장조사에 착수. 핵심 혐의는 입점 브랜드의 경쟁 플랫폼 거래를 부당 제한('멀티호밍 제한')하고 최혜대우를 요구했는지 여부(대상 행위는 2019년 이후). → **확신도 상**(정부기관 조치 보도, 조사 단계).
- **문제 3-1 — 직매입 거래 '갑질' 의혹 (계약 측면)**: 공정위가 2026년 4월 27일 무신사 본사에 조사관을 파견해 대규모유통업법 위반 의혹으로 현장조사. 조사 대상 행위에 판촉비용 전가, 경영정보 제공 요구, 대금 정산 지연 포함. 무신사 2025년 매출 구성상 직매입(상품) 비중 27.3%로 상당. → **확신도 상**(정부기관 조치를 복수 언론이 보도, 조사 단계).

### 플러그인이 하는 일

입점·납품 계약서(텍스트/PDF 추출본)를 입력받아, 대규모유통업법·공정거래법상 문제 소지 조항을 자동 탐지한다. 탐지 항목: ① 경쟁 플랫폼 거래를 제한하는 배타조건부 조항(멀티호밍 제한), ② 최혜대우(다른 채널 대비 동등·우대 가격 강제) 조항, ③ 판촉비·수수료를 납품업체에 전가하는 조항, ④ 부당 반품·경영정보 요구 조항.

### Codex 구성

`SKILL.md`(조항 유형별 탐지 규칙 + 근거 법조문 매핑 + 리스크 등급 기준) 중심. 계약서 파싱용 실행 코드(정규식·섹션 분할) 추가. **산출물 = 위험 조항 목록 + 인용 원문 + 근거 법조문 + 리스크 등급(상/중/하) + 대체 문구 제안**.

### 한계

최종 위법성 판단은 공정위·법률자문 영역이며, 플러그인은 '검토 우선순위 큐'까지만 제공한다(법률 의견 대체 아님).
