# logs/

이 폴더에는 플러그인을 만들면서 AI와 주고받은 대화 로그가 **원본 그대로** 저장됩니다.

- Claude Code / Codex CLI 사용 시: `log-hooks.zip`(예선 안내의 로그 수집 도구)을 작업 폴더 루트에 압축 해제하면 훅이 이 폴더에 로그를 자동 저장합니다.
- 그 밖의 도구 사용 시: 전체 대화를 편집·발췌 없이 txt/md/json 형식으로 이 폴더에 넣습니다.

⚠️ 로그는 **사후 가공(편집·발췌·삭제) 없이** 제출해야 합니다. API 키·비밀번호·토큰 등 비밀정보는 대화에 입력하지 마세요.
